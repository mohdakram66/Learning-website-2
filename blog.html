Responsive Navbar कैसे बनाएं (Hindi)

समस्या: Desktop पर अच्छा, mobile पर टूटा हुआ menu।

समाधान: CSS Grid + JS hamburger। Code:

nav { display: grid; grid-template-columns: 1fr auto; }
@media (max-width:768px) { .menu { display: none; } .hamburger { display: block; } }

फायदे: 2KB size, no library। Live demo ऊपर।
